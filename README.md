# Instagram IFrame Creator

## Install

1. Clone the project with: `$ git clone https://github.com/kaanersoy/vue-instagram-iframe-creator.git`
2. Run the server like:
```bash
//for deploy
npm run start 
//for developement
npm run dev
```
